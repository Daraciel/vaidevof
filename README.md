# VAIDEVOF — Valkyr AI Development Office

Stack de desarrollo asistido por IA usando:

- ZeroClaw
- PicoClaw
- Ollama
- Agentes especializados (Spec, Dev, Design, Reviewer, Orchestrator)

## Estructura

- `docker-compose.yml` — Infraestructura del stack
- `agents/` — Agentes en YAML
- `project/` — Proyecto gestionado por los agentes
